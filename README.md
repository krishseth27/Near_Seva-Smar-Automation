# Near_Seva-Smar-Automation
NearSeva is a lightweight and user‑friendly web application designed to help daily service providers—like tiffin vendors, milk delivery agents, paper delivery workers, and similar services—manage their deliveries efficiently.
The platform reduces manual calling, avoids confusion, and brings transparency for both service providers and customers.

--Features

-Smart Notification Flow
“Near Customer” alert triggered by provider
Customer instantly gets a notification pop‑up
Provider sees a delivery‑marking popup

-Daily Delivery Tracking
Delivered / Skip / Not Available status
Auto-updated calendar view
Clean color-coded UI

-Skip & Vacation Mode
Customer can skip the next day
Customer can set a vacation date range
Provider’s route list updates automatically

-Automated Billing
Auto-calculates monthly bill based on actual delivery days
Completely transparent for both sides

-Dual Language Support
English/Hindi
Switch language instantly from the UI.

-Separate Dashboards
Provider Dashboard: Customer list, delivery status, alerts
Customer Dashboard: Calendar, billing, skip/vacation options

--Future Enhancements
Real GPS-based near-customer detection

Persistent database (MongoDB)
Authentication system
UPI payment integration
